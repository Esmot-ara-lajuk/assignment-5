What is the Document Object Model (DOM)?

The Document Object Model is a programming interface for web documents.The DOM was designed to be independent of any particular programming language.

How do you select an element from the DOM?

We can select DOM by using getElementById, getElementsByClassName, getElementsByTagName, querySelector etc.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in JavaScript is a pattern that efficiently handles events.Instead of attaching event listeners to multiple child elements, you can add a single event listener to a parent element.

How do you manipulate an element's attributes and styles using the DOM?

Change the content of an element, manipulate the class attributes, setting CSS styles using JavaScript, creating adding and removing elements etc by using these DOM elements attributes can be manipulate.
